GAMS Routine Finder

Use this hypertext file to navigate through the GAMS classification scheme for mathematical software and locate desired SLATEC routines. Links on the right (i.e., on the descriptions) take you to finer levels of detail. Links on the left (i.e., on the alphanumerical codes) return you to coarser levels of the hierarchy.

Links named "SLATEC" will take you to the corresponding point in the Table of Contents of the SLATEC Common Mathematical Library. Note that not all GAMS Categories are represented in SLATEC, so SLATEC links are not always displayed.
