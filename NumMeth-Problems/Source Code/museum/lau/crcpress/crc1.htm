A Numerical Library in C for Scientists and Engineers

Author - H.T. Lau

Concordia University and Bell Northern Research, Montreal, Quebec

DESCRIPTION

This extensive library of computer programs-written in C language-al lows readers to solve numerical problems in areas of linear algebra, ordinary an d partial differential equations, optimization, parameter estimation, and specia l functions of mathematical physics.

The library is based on NUMAL, the program assemblage developed and used at t he Centre for Mathematics and Computer Science in Amsterdam, one of the world's leading research centers. The important characteristic of the library is its mod ular structure. Because it is highly compact, it is well-suited for use on perso nal computers.

The library offers the expert a prodigious collection of procedures for imple menting numerical methods. The novice can experiment with the worked examples pr ovided and use the more comprehensive procedures to perform mathematical computa tions. The library provides a powerful research tool for computer scientists, en gineers, and applied mathematicians.

FEATURES

  • Provides a high quality, inexpensive mathematical library
  • Offers a highly compact modular structure particularly suited to implementat ion on a personal computer
  • Uses the ANSI C standard throughout to ensure portability
  • Includes 130 pages of worked examples
